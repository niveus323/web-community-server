# web-community-server
SpringBoot 게시판 미니프로젝트

### 1. 사용 기술
- Spring-Boot (SDK 11)
- JPA
- H2, Mysql
- Thymeleaf
- BootStrap (v.3.3.7)
- OAuth2
- Spring-Boot Data Rest
- Websocket
- Spring Security

### 2. 주요 기능
- 게시물, 댓글 CRUD
- 게시물 추천 기능
- JPA 페이징 및 댓글 더보기 기능
- 게시글 검색 기능
- Security, Login API를 활용한 접근 제어
- WebSocket을 활용한 댓글 추가, 게시물 추천 시 알림
